# Light Fish

A tiny php framework base on slim.


## Config

### Routes

The routes config file in: 

```
/routes/web.php
```
Read more on [Slim Doc](http://www.slimframework.com/docs/objects/router.html)

### Controller 

All Controller in

```
/App/Controllers
```

### Views
 
using twig.
